# The point of this file is to replace all instances of "culture:<CULTURE>", or "religion:<RELIGION>",
# or "faith:<FAITH>" in the database with an effect; this means that submods merely have to overwrite
# trigger definitions this file to tweak the applicability of RICE events, such as, say, splitting
# Pamiri or Sami into sub-cultures or applying Zoroastrian events to a fantasy religion/faith
#
#All these triggers exist at culture/faith/religion scope
